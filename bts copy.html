A mix of CSS grid and flex

The main structure of the site is made using a grid template. Grid lines are named for ease of reference. minmax(32px, auto) defines the space between the the left/right edges of the site and the vertical lines that borders the content. This space has a minimum width of 32px and grows with the width of the screen. Similarly, the main content area is set to not grow beyond 1280px. Using media query, the left/right margins will be reduced when viewing the site on a narrower screen.

.container{
margin: 0 0;
height: 100%;
display: grid;
grid-template-columns: [border-start] minmax(32px, auto) [main-start] minmax(auto, 1280px) [main-end] minmax(32px, auto) [border-end];
grid-template-rows: [border-start] 100px
                    [title-start] auto [title-end] var(--section_padding)
                    [sec1-start] 3fr [sec1-end] var(--section_padding)
                    [sec2-start] 3fr [sec2-end] var(--section_padding)
                    100px [border-end];
}
@media (max-width: 400px) {
.container{
    grid-template-columns: [border-start] 16px [main-start] minmax(auto, 368px) [main-end] 16px [border-end] ;
    grid-template-rows: [border-start] 100px
                        [title-start] auto [title-end] var(--section_padding)
                        [sec1-start] 3fr [sec1-end] var(--section_padding)
                        [sec2-start] 3fr [sec2-end] var(--section_padding)
                        100px [border-end];

}
}

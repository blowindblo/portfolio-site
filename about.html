Website Structure and Design

The main structure of this site is built using a combination of CSS Grid and Flexbox. Grid lines are named for ease of reference. The left and right margins of the site is defined by minmax(32px, auto), ensuring a minimum width of 32px that expands with the screen size. The main content area is constrained to a maximum width of 1280px. Media queries are used to adjust the left and right margins for optimal viewing on narrower screens. 

            .container{
            margin: 0 0;
            height: 100%;
            display: grid;
            grid-template-columns: [border-start] minmax(32px, auto) [main-start] minmax(auto, 1280px) [main-end] minmax(32px, auto) [border-end];
            grid-template-rows: [border-start] 100px
                                [title-start] auto [title-end] var(--section_padding)
                                [sec1-start] 3fr [sec1-end] var(--section_padding)
                                [sec2-start] 3fr [sec2-end] var(--section_padding)
                                100px [border-end];
            }
            @media (max-width: 400px) {
            .container{
                grid-template-columns: [border-start] 16px [main-start] minmax(auto, 368px) [main-end] 16px [border-end] ;
                grid-template-rows: [border-start] 100px
                                    [title-start] auto [title-end] var(--section_padding)
                                    [sec1-start] 3fr [sec1-end] var(--section_padding)
                                    [sec2-start] 3fr [sec2-end] var(--section_padding)
                                    100px [border-end];
